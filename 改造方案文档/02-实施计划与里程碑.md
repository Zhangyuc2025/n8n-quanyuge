## 🚨 重大调整说明

**发现问题：** 删除节点后前端崩溃，无法查看工作流编辑器
**根本原因：** package.json 硬编码节点引用 → 后端加载失败 → 前端未做容错
**解决方案：** 必须先实现节点动态化架构，后端验证元数据，前端从 API 加载
**优先级调整：** 节点动态化（阶段 5.2-5.4）提到最高优先级，中文化延后
